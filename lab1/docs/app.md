<h1>Документація по розробленому web-застосунку</h1>

<h2>Технологічний стек</h2>

- ASP.NET 10
- Entity Framework Core
- PostgreSQL

<h2>API - endpoints</h2>

Службові endpoints

| Endpoint | Опис |
--- | ---
| GET / | повертає HTML-сторінку з переліком усіх доступних ендпоінтів |
| GET /health/alive | повертає 200, якщо процес запущено |
| GET /health/ready | повертає 200, якщо успішно підключено до БД, інакше 500 |

<br>

Endpoints бізнес-логіки
| Endpoint | Опис |
--- | ---
| POST /notes | створює нову нотатку |
| GET /notes | повертає список усіх нотаток |
| GET /notes/{id} | повертає нотатку за ідентифікатором |
